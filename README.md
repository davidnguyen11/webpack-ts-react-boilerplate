# webpack typesript react boilerplate

## Development

**Watch**

**yarn**

```bash
yarn install
```

```bash
yarn start
```

**npm**

```bash
npm i
```

```bash
npm start
```

**Build**

**yarn**
```bash
yarn install
```

```bash
yarn build
```

**npm**

```bash
npm i
```

```bash
npm run build
```
